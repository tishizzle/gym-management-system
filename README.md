
# Gym Management and Fitness Tracking System

## Overview

A gym management system developed using Python and SQL to manage members, trainers, workout plans, attendance records, and performance tracking.

## Features

- Member management
- Trainer management
- Workout tracking
- Attendance tracking
- Performance reports
- Database integration using SQL
- Python GUI for user interaction

## Technologies Used

- Python
- SQL
- Database Management Systems

## Key Functionalities

- Store and manage member records
- Track trainer assignments
- Generate member performance reports
- Monitor daily, weekly, and monthly activity
- Retrieve data through SQL queries

## Learning Outcomes

This project strengthened my understanding of:

- Database design
- SQL querying
- Python-SQL integration
- Data organization and reporting
- Application development
